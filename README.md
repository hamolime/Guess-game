# 🎯 NumMaster - Number Guessing Game

A fun and interactive number guessing game built using **HTML, CSS, and JavaScript**.  
Try to guess the secret number in the fewest attempts possible!

---

## 🚀 Features

- 🎲 Random number generation
- 💡 Smart hints (Too high / Too low)
- 🧠 Simple and addictive gameplay
- 📱 Responsive design (works on mobile & desktop)
- ⚡ Fast and lightweight

---

## 🎮 How to Play

1. The system generates a random secret number.
2. Enter your guess in the input box.
3. Receive hints after each attempt:
   - Too high 📈
   - Too low 📉
4. Keep guessing until you find the correct number.

---
